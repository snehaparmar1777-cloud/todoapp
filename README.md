# todoapp
# 🔧 Interactive Todo App - AngularJS SPA

## ✨ Features
- ➕ **Add** tasks (Enter or Add button)
- ✏️ **Edit** inline (double-click text)
- ✅ **Toggle** complete (strikethrough)
- 🔍 **Search** + **Filter** (All/Active/Completed)
- 🗑️ **Remove** with **Undo** (5-level stack)
- 📊 **Active count** display
- 💾 **localStorage** persistence

## 🚀 Quick Start
```
1. Open index.html in browser
2. No setup required!
```

## 📱 Demo
```
Add tasks → Edit → Complete → Filter → Undo → Clear
Data persists on refresh
```
